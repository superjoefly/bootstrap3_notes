Bootstrap Dropdowns

A dropdown menu is a toggleable menu that allows a user to choose one value from a predefined list:

Basic Dropdown

The .dropdown class indicates a dropdown menu.

The .dropdown-toggle class and the .data-toggle="dropdown" attribute are used on a button or link to open and close the dropdown menu.

The .caret class creates a down arrow to indicate that the button provides a dropdown option.

The .dropdown-menu class is used on an ul-element to build the dropdown menu.

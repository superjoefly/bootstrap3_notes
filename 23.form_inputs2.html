Form Control States

  • Input Focus - outline is removed and a box-shadow is applied on focus
  • Disabled - add a disabled attribute to disable input field
  • Disabled Fieldsets - add a disabled attribute to fieldset to disable all controls within
  • Readonly Inputs - prevent user input
  • Validation States - validation styles for error, warning, and success messages. Requires the .has-warning, .has-error, or .has-success classes on the parent element
  • Icons - add feedback icons with the .has-feedback class and an icon
  • Hidden Labels - add a sr-only class on non-visible labels

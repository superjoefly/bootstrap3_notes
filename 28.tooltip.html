Bootstrap Tooltip

The tooltip plugin is a small popup box that appears when a user moves the mouse pointer over an element:

Create a Tooltip

To create a tooltip, use the data-toggle="tooltip" attribute on an element.

Use the title attribute to specify the text that should be displayed inside the tooltip:

Hover Over Me!

Positioning Tooltips

By default, the tooltip appears on top of the element.

Use the data-replacement attribute to position the tooltip on top, bottom, left or right of the element:

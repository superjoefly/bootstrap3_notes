Media Objects

Bootstrap provides a way to align media objects, like images or video, to the left or to the right of some content.

Use a div-element with the .media class to create a container for media objects.

Use the .media-left class to align the media object (image) to the left, or the .media-right class to align the media object to the right.

Text that should appear next to the media should be placed inside a container with class="media-body".

You can also use .media-heading for headings.

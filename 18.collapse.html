Bootstrap Collapse

Collapsibles are useful when you want to hide and show large amounts of content:

Basic Collapsible

Use the .collapse class to indicate a collapsible element.

Use the data-toggle="collapse" attribute on a link or a button to show/hide the content.

Use the data-target="#id" attribute to connect the button/link with the content.

Note: for a-elements, you can use the href attribute instead of the data-target attribute.

Responsive Images

Responsive images automatically adjust to fit the size of the screen.

Use the .img-responsive class on the image tag so that it will scale according to it's parent element.

Responsive Embeds

Use the .embed-responsive-item class on these elements: iframe, embed, video, object. This allows video and/or slideshows to scale properly.

The containing div defines the aspect ratio (16by9).

The two valid aspect ratio options are 16by9 and 4by3.
